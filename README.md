# 🌐 Safe Horizon – Smart Disaster Management Platform

Safe Horizon is an AI-powered disaster management platform designed to enhance **preparedness, real-time response, and recovery** during natural and man-made disasters.

It integrates **weather alerts, emergency contacts, AI chatbot support, donation system, and volunteer coordination** into one unified platform.

---

## 🚀 Features

### 🌩️ Real-Time Weather Alerts
- Get live weather data using OpenWeather API
- Select city or use current location
- Displays temperature, wind speed, and humidity

### 🤖 AI Chatbot (HoriChat)
- Built using **NLTK + TensorFlow + FastAPI**
- Handles queries like:
  - Emergency contacts
  - Disaster help
  - Navigation assistance
- Smart intent-based responses

### 📞 Emergency Contacts
- Quick access to:
  - Police (100)
  - Ambulance (102)
  - Fire (101)
  - National Emergency (112)
- Includes specialized helplines

### 🤝 Donation System
- Users can:
  - Donate money (UPI, card, etc.)
  - Request donations
- Helps support disaster victims

### 👥 Volunteer Connect
- Connect victims with nearby volunteers
- Community-based disaster response

### 📊 Interactive Dashboard
- Clean UI for:
  - Alerts
  - Information
  - Navigation

### 🔐 Authentication System
- Login & Signup functionality
- Google Sign-In integration

### 📩 Contact & Feedback
- Users can send feedback or queries
